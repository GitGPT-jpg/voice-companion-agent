# VBF

VirtualBF is an AI companion app prototype with chat, TTS, singing, memory, and both Web and desktop interfaces.

## Included

- Python backend and orchestration code
- Web templates and frontend assets
- Example environment file and dependency list

## Excluded from this public repo

- Local `.env` secrets
- Model weights and indexes
- Song assets and generated audio
- Chat logs, memory data, and runtime caches

## Quick start

1. Create a virtual environment.
2. Install dependencies with `pip install -r requirements.txt`.
3. Copy `.env.example` to `.env` and fill in your API keys.
4. Run `python web_app.py` for the Web version or `python gui.py` for the desktop version.
